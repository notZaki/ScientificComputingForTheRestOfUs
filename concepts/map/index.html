Map, filter, reduce

Map, filter, reduce: start the lesson
Reading time: 0 minutes
Status: construction
Key concepts: map filter reduce
A lot of computing tasks involve going through arrays, or collections, of values, and selecting some of them, applying transformations, and returning values based on them. After reading the lesson on Control flow, you are equipped to do this using for, if, and while. But most tasks can actually be adressed using an approach which is even more simple: map/filter/reduce.
